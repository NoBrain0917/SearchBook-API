# SearchBook-API

Based on [독서교육종합지원시스템](http://reading.ssem.or.kr/)    
First node.js API work



# Usage
`http://localhost:3000/?book=책이름&school=학교이름&local=지역이름`

# Example
```
http://localhost:3000/?book=코딩&school=고등학교&local=서울
```
